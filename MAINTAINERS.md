# Maintainers

This project is maintained by:

- **user6m** ([@user6m](https://github.com/user6m))

## Responsibilities

Maintainers are responsible for:

- Reviewing pull requests.
- Merging approved pull requests into the `main` branch.
- Creating and publishing new releases.
- Triaging issues and ensuring the project remains healthy.

## Becoming a Maintainer

Contributors who have shown a strong commitment to the project by consistently submitting high-quality pull requests and helping with issue triage may be invited to become maintainers.
