## Description

An interactive CLI tool to delete local git branches safely.

## Setup

This project uses `pnpm`. Ensure you have [Corepack](https://nodejs.org/api/corepack.html) enabled.

```bash
corepack enable pnpm
pnpm install
```

## Development

```bash
# Typecheck
pnpm run typecheck

# Build once
pnpm run build

# Build for development (with sourcemaps)
pnpm run dev

# Lint
pnpm run lint
```
